## Run Locally

Clone the project

```bash
  git clone https://github.com/itjewel/ThreeJS-React-3D-Tshart-AI-dalle.git
```


Install dependencies
Start the client
```bash
   cd client
   npm i
   npm rund dev
```

Start the server

```bash
   cd server
   npm i
   npm start
```
To install a new package

```bash
  npm i <package_name>
```
To remove a package

```bash
  npm uninstall <package_name>
```
